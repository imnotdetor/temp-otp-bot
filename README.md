# Temp OTP Telegram Bot

This Telegram bot provides virtual numbers for OTP verification.
Users can:
- View profile & balance
- Buy virtual numbers
- Receive OTP codes
- Deposit balance
- Earn via referrals

### Tech Stack
- Python
- python-telegram-bot
- Railway Hosting

### Setup
1. Add BOT_TOKEN & ADMIN_ID in Railway Variables
2. Deploy project
3. Bot starts automatically

⚠️ This is a test/demo bot with hardcoded numbers & OTP.
